This page contains reference information about deco.chat.

## Quick Reference

### CLI Commands

- `deco login` - Authenticate with deco.chat
- `deco init` - Create a new project
- `deco deploy` - Deploy to Cloudflare Workers
- `deco gen` - Generate types for integrations
- `deco gen:self` - Generate types for your own tools

### Project Structure

```
my-deco-app/
├── server/          # Cloudflare Workers code
│   ├── main.ts      # Tools and workflows
│   └── wrangler.toml
└── view/            # React frontend (optional)
    └── src/
```

### Key Concepts

- **Tools**: Single functions that perform specific tasks
- **Workflows**: Multi-step sequences that orchestrate tools
- **Integrations**: Pre-built connections to external services
- **Agents**: AI actors that can use tools and workflows
