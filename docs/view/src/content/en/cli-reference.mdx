Essential deco CLI commands for project management and deployment.

## Authentication

```bash
deco login      # Authenticate with deco.chat
deco logout     # Log out of CLI
deco whoami     # Check current account
```

## Project Management

```bash
deco create     # Create new project from template
deco configure  # Link project to workspace
```

## Development

```bash
npm run dev                              # Start local development
deco link                                # Expose local server to public URL
```

Use `deco link` to connect external AI clients (Cursor, Claude Desktop) to your local MCP server.

## Deployment

```bash
deco deploy     # Deploy to Cloudflare Workers
```


## Utilities

```bash
deco update     # Update CLI to latest version
deco --help     # Show all commands
```
