Comandos essenciais da CLI deco para gerenciamento de projetos e deployment.

## Autenticação

```bash
deco login      # Autenticar com deco.chat
deco logout     # Fazer logout da CLI
deco whoami     # Verificar conta atual
```

## Gerenciamento de Projetos

```bash
deco create     # Criar novo projeto a partir de template
deco configure  # Vincular projeto ao workspace
```

## Desenvolvimento

```bash
npm run dev                              # Iniciar desenvolvimento local
deco link                                # Expor servidor local para URL pública
```

Use `deco link` para conectar clientes IA externos (Cursor, Claude Desktop) ao seu servidor MCP local.

## Deployment

```bash
deco deploy     # Deploy para Cloudflare Workers
```


## Utilitários

```bash
deco update     # Atualizar CLI para versão mais recente
deco --help     # Mostrar todos os comandos
```
