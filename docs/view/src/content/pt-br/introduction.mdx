**deco.chat é uma base open-source para construir software nativo de IA.**

Crie servidores MCP que expõem ferramentas e workflows para agentes de IA, com interfaces web customizadas opcionais, tudo deployado na rede global edge da Cloudflare.

## Para quem é?

- **Vibecoders** prototipando ideias
- **Agentic Engineers** fazendo deploy de sistemas de produção escaláveis e
  seguros

## Por que deco.chat?

Nosso objetivo é simples: empoderar times com IA Generativa dando aos builders
as ferramentas para criar aplicações de IA que escalam além da demo inicial e
chegam a milhares de usuários, de forma segura e custo-efetiva.

## Capacidades principais

- **Runtime Open-source** – Componha facilmente ferramentas, workflows e views dentro de
  um único codebase
- **MCP Mesh (Model Context Protocol)** – Integre com segurança modelos, fontes de
  dados e APIs, com observabilidade e controle de custos
- **Stack TypeScript Unificado** – Combine lógica backend e frontends React/Tailwind
  customizados de forma fluida usando RPC tipado
- **Infraestrutura Global e Modular** – Construído na Cloudflare para deployments
  de baixa latência e infinitamente escaláveis. Self-host com sua Cloudflare API Key
- **Workspace Visual** – Construa agentes, conecte ferramentas, gerencie permissões e
  orquestre tudo construído em código

## Conceitos Fundamentais

Antes de aprofundar, vamos definir rapidamente alguns conceitos fundamentais:

### Tools
Funções que agentes podem chamar para realizar ações como chamadas de API, consultas de banco de dados ou computações.

```typescript
const createCepSearchTool = (env: Env) =>
  createTool({
    id: "CEP_SEARCH",
    description: "Search Brazilian postal codes",
    inputSchema: z.object({ cep: z.string() }),
    outputSchema: z.object({ city: z.string(), state: z.string() }),
    execute: async ({ inputData }) => {
      // Tool implementation
    },
  });
```

### Workflows
Sequências determinísticas que orquestram múltiplas ferramentas para automação.

```typescript
const createCustomerWorkflow = (env: Env) =>
  createWorkflow({ id: "REGISTER_CUSTOMER" })
    .then(validateInput)
    .then(saveToDatabase) 
    .then(sendWelcomeEmail)
    .commit();
```

### MCP (Model Context Protocol)
O padrão que conecta modelos de IA às suas ferramentas. Seu Cloudflare Worker serve como um servidor MCP que agentes podem descobrir e usar.

### Agentes
Assistentes de IA configurados com acesso às suas ferramentas. Eles entendem linguagem natural e decidem quando chamar ferramentas específicas para ajudar usuários.

### Views
Frontends React opcionais que fornecem interfaces customizadas para suas ferramentas e workflows.
