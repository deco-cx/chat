**Deco é uma base open-source para construir software nativo de IA.**

Crie servidores MCP que expõem ferramentas e fluxos de trabalho para agentes de IA, com interfaces web customizadas opcionais, tudo implantado na rede global edge da Cloudflare.

## Para quem é?

- **Vibecoders** prototipando ideias
- **Engenheiros agênticos** implantando sistemas de produção escaláveis, seguros e sustentáveis

## Por que Deco?

Nosso objetivo é simples: capacitar equipes com IA Generativa, dando aos construtores as ferramentas para criar aplicações de IA que escalam além da demonstração inicial e chegam a milhares de usuários, de forma segura e econômica.

## Capacidades principais

- **Runtime Open-source** – Facilmente componha ferramentas, fluxos de trabalho e visualizações dentro de uma única base de código
- **MCP Mesh (Protocolo de Contexto do Modelo)** – Integre com segurança modelos, fontes de dados e APIs, com observabilidade e controle de custos
- **Stack TypeScript Unificado** – Combine lógica de backend e frontends React/Tailwind customizados perfeitamente usando RPC tipado
- **Infraestrutura Global e Modular** – Construído na Cloudflare para implantações de baixa latência e infinitamente escaláveis. Auto-hospede com sua Chave de API da Cloudflare
- **Workspace Visual** – Construa agentes, conecte ferramentas, gerencie permissões e orquestre tudo construído em código

## Conceitos Principais

Antes de mergulhar, vamos definir rapidamente alguns conceitos fundamentais:

### Ferramentas
Funções que podem ser chamadas por agentes, fluxos de trabalho, visualizações e outras ferramentas para executar ações como chamadas de API, consultas de banco de dados ou computações.

```typescript
const createCepSearchTool = (env: Env) =>
  createTool({
    id: "CEP_SEARCH",
    description: "Pesquisar códigos postais brasileiros",
    inputSchema: z.object({ cep: z.string() }),
    outputSchema: z.object({ city: z.string(), state: z.string() }),
    execute: async ({ inputData }) => {
      // Implementação da ferramenta
    },
  });
```

### Fluxos de Trabalho
Sequências determinísticas que orquestram múltiplas etapas, que frequentemente são ferramentas, para criar automações.

```typescript
const createCustomerWorkflow = (env: Env) =>
  createWorkflow({ id: "REGISTER_CUSTOMER" })
    .then(validateInput)
    .then(saveToDatabase) 
    .then(sendWelcomeEmail)
    .commit();
```

### MCP (Protocolo de Contexto do Modelo)
O padrão que conecta modelos de IA às suas ferramentas. Seu Cloudflare Worker serve como um servidor MCP que agentes podem descobrir e usar.

### Agentes
Assistentes de IA configurados com acesso às suas ferramentas. Eles entendem linguagem natural e decidem quando chamar ferramentas específicas para ajudar usuários.

### Visualizações
Frontends React opcionais que fornecem interfaces customizadas para suas ferramentas e fluxos de trabalho.
