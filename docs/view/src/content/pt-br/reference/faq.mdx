## Perguntas sobre Desenvolvimento

<AccordionGroup>
  <Accordion title="Como adiciono uma nova tool?" icon="Wrench" client:load>
    1. Defina a tool em `server/main.ts` usando `createTool`
    2. Adicione-a ao array tools do `withRuntime`
    3. Execute `npm run gen` para atualizar tipos
    4. Teste via UI do deco ou cliente RPC
  </Accordion>

  <Accordion title="Como me conecto a uma API externa?" icon="Globe" client:load>
    1. Adicione a integração via dashboard do deco
    2. Execute `npm run gen` para obter interfaces tipadas
    3. Use `env.INTEGRATION_NAME.method()` nas suas tools
  </Accordion>
</AccordionGroup>

## Perguntas Técnicas

<AccordionGroup>
  <Accordion title="Qual é a diferença entre tools e workflows?" icon="Settings" client:load>
    Tools são funções únicas que executam tarefas específicas. Workflows orquestram múltiplas tools com lógica, condições e loops.
  </Accordion>

  <Accordion title="Posso usar meu próprio banco de dados?" icon="Database" client:load>
    Sim. Você pode usar Cloudflare D1, Workers KV, ou conectar a bancos de dados externos como 
    Supabase ou Planetscale via APIs HTTP.
  </Accordion>
</AccordionGroup>

## Perguntas sobre Integrações

<AccordionGroup>
  <Accordion title="Quais integrações estão disponíveis?" icon="Link" client:load>
    Muitos serviços comuns como Slack, Gmail, Notion, Stripe, e mais. Verifique o dashboard do deco para a lista completa.
  </Accordion>

  <Accordion title="Posso criar integrações personalizadas?" icon="Code" client:load>
    Sim, você pode escrever tools personalizadas que chamam qualquer API HTTP usando `fetch`.
  </Accordion>
</AccordionGroup>

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Minha tool não está funcionando" icon="AlertCircle" client:load>
    1. Verifique se está registrada em `withRuntime`
    2. Verifique seus schemas Zod
    3. Verifique os logs para erros
    4. Certifique-se de que executou `npm run gen` após mudanças
  </Accordion>

  <Accordion title="Meu workflow não está executando" icon="AlertTriangle" client:load>
    1. Certifique-se de que chamou `.commit()` no final
    2. Verifique se todos os passos estão adequadamente encadeados
    3. Verifique se seus schemas de input/output coincidem
  </Accordion>
</AccordionGroup>
